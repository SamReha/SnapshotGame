AUDIO README

To put audio in the scene, drop the 'Audio' prefab into the scene.

AVAILABLE FUNCTIONS:

AudioManager.getInstance().setDeerVolume (float value);
AudioManager.getInstance().setFoxVolume (float value);
AudioManager.getInstance().setClearVolume float value);
AudioManager.getInstance().setRainVolume (float value);
AudioManager.getInstance().setNoon();
AudioManager.getInstance().setDawn();
